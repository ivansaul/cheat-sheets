Welcome to Cheat Sheets. This page informs you of our policies regarding the collection, use, and disclosure of personal information when you use our application and the services associated with it. By using the Cheat Sheets application, you agree to the practices described in this privacy policy.

## 1. Information We Collect

### 1.1 Non-Personal Data

Cheat Sheets is an open-source application, meaning we do not collect personally identifiable information (PII) or any sensitive data. We do not actively track user activity or gather personally identifiable data.

### 1.2 Locally Generated Data

Cheat Sheets may generate data locally to enhance the user experience, such as locally storing preferences and settings. However, this data is not shared or transmitted outside the application.

## 2. Use of Information

### 2.1 User Experience Enhancement

Any locally generated information is used solely to enhance the user experience within the application. We do not share this information with third parties.

## 3. Sharing Information

### 3.1. Open Source Code

Cheat Sheets is an open-source application, and all source code is available on [GitHub](https://github.com/ivansaul/cheat-sheets). Transparency is fundamental, and anyone can review, contribute to, and audit the code.

### 3.2. Open Collaboration

Being an open-source project, collaboration is welcomed. However, user information or personal data is not shared in the collaboration process.

## 4. Security

### 4.1. Source Code Security

We are committed to maintaining the security of the source code. The open-source community can contribute to improving security through audits and suggestions.

## 5. Changes to the Privacy Policy

### 5.1. Updates on GitHub

Any changes to the privacy policy will be reflected in the GitHub repository. Users can review the change history at any time.

## 6. Contact

If you have questions or concerns about this privacy policy, feel free to contact us via email at ivansaul.dev@gmail.com.

By using the Cheat Sheets application, you agree to the terms and conditions outlined in this privacy policy.

Thank you for choosing Cheat Sheets.
