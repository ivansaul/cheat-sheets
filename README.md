## Building from Source

```bash
git clone https://github.com/ivansaul/cheat-sheets.git
cd cheat-sheets
flutter pub get
dart run build_runner build
flutter run --release
```

> [!TIP]
> For full local setup, backend, and environment configuration, see [**`DEVELOPMENT GUIDE`**](./DEVELOPMENT.md).
